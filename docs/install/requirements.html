RequirementsÂ¶

Envoy was initially developed and deployed on Ubuntu 14 LTS. It should work on any reasonably recent Linux including Ubuntu 16 LTS.

Envoy has the following requirements:

  • GCC 4.9+ (for C++11 regex support)
  • cotire (last tested with 1.7.8)
  • spdlog (last tested with 319a62)
  • http-parser (last tested with 2.7.0)
  • nghttp2 (last tested with 1.9.2)
  • libevent (last tested with 2.0.22)
  • tclap (last tested with 1.2.1)
  • gperftools (last tested with 2.5.0)
  • jansson (last tesed with 2.7)
  • openssl (last tesed with 1.0.2h)
  • protobuf (last tested with 3.0.0)

In order to compile and run the tests the following is required:

  • googletest

In order to run code coverage the following is required:

  • gcovr
